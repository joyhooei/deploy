# 更新文档

## 一、手动更新

进入到项目路径，依次运行下面的命令：

```
git pull
```
> 拉取最新的项目文件

```
gitbook build
```
> 编译文件，编译后生成的静态文件会放置在 `_book` 目录。

```
rm -rf _book_cache
cp -r _book _book_cache
```
> 将编译的文件从 `_book` 复制到 `_book_cache`，最终网站运行时是读取 `_book_cache` 目录下的文件。
